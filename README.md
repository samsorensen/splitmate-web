# SplitMate Website

Marketing website for SplitMate, the iPhone app that makes shared expenses simple and fair.
Snap a receipt, speak an expense, or add it manually, and SplitMate instantly organizes who owes what with clear per-person breakdowns.

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dev site. 
Or open [http://split-mate.store](http://split-mate.store) to view the live site.
